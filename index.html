Variable declaration

Variables are a good way to set data that are required by the page. To create a single variable, use this:
@var $variable = value;
And to create multiple variables on a single line, use this:
@var $variable_1 = value_1; @var $variable_2 = value_2;

Note: every variable must end with a semi-colon (;)
